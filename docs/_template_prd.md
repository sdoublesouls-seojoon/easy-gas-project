# PRD 템플릿

> 새로운 기능 요청 시 이 템플릿으로 `docs/{title}_prd.md` 파일을 생성한다.

---

# {제목}

## 생성일
YYYY-MM-DD

## 목표
사용자가 달성하고자 하는 것을 한 문장으로.

## 요구사항

### 필수 (Must Have)
- [ ] 

### 선택 (Nice to Have)
- [ ] 

## 입력
- 어디서 데이터를 가져오는지
- 트리거 조건 (수동 실행 / 시간 기반 / 이벤트 기반)

## 출력
- 결과물이 어디에 저장/전송되는지

## 제약조건
- 실행 빈도
- 데이터 양
- 보안 요구사항

## 참고
- 관련 문서, 링크, 예시
